# Alarm Sesi

Bu klasöre `alarm.mp3` dosyasını ekleyin.

## Nasıl Çalışır?

- Dashboard'da bir acil durum alarmı geldiğinde (`emergency-alert` socket eventi) otomatik olarak çalar.
- Banner'daki **"🔇 Sesi Kapat"** butonu ile durdurulabilir.
- Dosya olmadan ses çalmaz, ancak diğer özellikler (banner, pinli alarmlar) çalışmaya devam eder.

## Dosya Gereksinimleri

- **Format:** MP3 (önerilen) veya WAV
- **Dosya adı:** `alarm.mp3` (tam bu isimle, büyük/küçük harf duyarlı)
- **Süre:** 5-30 saniye önerilir (döngü modunda çalar)
